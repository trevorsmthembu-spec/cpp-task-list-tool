# C++ Task List Tool

A simple console-based task management application written in C++.

## Features

- Add new tasks
- View all tasks
- Remove tasks
- Dynamic task storage using `std::vector`
- Menu-driven interface
- Input validation for task removal

## Technologies Used

- C++
- Standard Template Library (STL)
- std::vector
- std::string
- Console I/O

## How to Run

1. Clone the repository

```bash
git clone https://github.com/trevorsmthembu-spec/cpp-task-list-tool.git
```

2. Open the project in Visual Studio Code or Visual Studio.

3. Compile the program

```bash
g++ main.cpp -o TaskList
```

4. Run

```bash
./TaskList
```
